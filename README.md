# Elegance Hair Salon - Website

A professional hair salon website featuring a modern, responsive design with multiple sections for showcasing services, gallery, and contact information.

## Project Structure

```
HairDresser/
├── index.html          # Main HTML file with all website sections
├── styles.css          # CSS stylesheet for styling and responsiveness
└── README.md           # This file
```

## Features

### Header & Navigation
- Sticky navigation bar for easy access to all sections
- Professional logo/branding
- Navigation links to all main sections:
  - Home
  - Services
  - Gallery
  - About Us
  - Contact

### Main Content Sections

1. **Hero Section**
   - Eye-catching banner with welcome message
   - Call-to-action button for booking

2. **Services Section**
   - Grid layout showcasing 4 main services:
     - Haircut
     - Hair Coloring
     - Styling
     - Hair Treatment
   - Hover animations on service cards

3. **Gallery Section**
   - Responsive grid for displaying portfolio images
   - Placeholder items ready for actual images

4. **About Section**
   - Business description and history

5. **Contact Section**
   - Phone, email, and physical address
   - Easy-to-read contact information

### Footer
- Company information
- Quick links
- Social media links
- Copyright notice

## Design Features

- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Modern Color Scheme**: Professional dark blue (#2c3e50) with accent red (#e74c3c)
- **Smooth Animations**: Hover effects on navigation and service cards
- **Clean Typography**: Easy-to-read fonts and proper spacing
- **Grid Layouts**: Modern CSS Grid for flexible content arrangement

## Getting Started

1. Open `index.html` in your web browser
2. Navigate through the sections using the menu
3. Click "Book Now" to trigger the booking action

## Customization

### To Change Business Name:
Edit `<h1>` in the header section in `index.html`

### To Update Services:
Modify the service cards in the Services section

### To Add Gallery Images:
Replace placeholder gallery items with actual image tags

### To Update Contact Information:
Edit the contact details in the Contact section

### To Customize Colors:
Edit the CSS variables in `styles.css`:
- Primary color: `#2c3e50` (dark blue)
- Accent color: `#e74c3c` (red)
